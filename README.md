# Facial Strain Detection and Adaptive Display Adjustment System

This project detects facial strain in real-time using computer vision and adjusts display settings automatically.

## Structure
- `data/`: Datasets (raw and processed)
- `models/`: Trained model files
- `scripts/`: Training and preprocessing scripts
- `real_time/`: Real-time detection application
- `utils/`: Helper functions
- `notebooks/`: Experiments and EDA
- `logs/`: Log files
